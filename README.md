## TCP  Сервер
### Простой TCP сервер парсит HTTP запросы и показывает их содержимое.
#### Установка
```
git clone https://github.com/dropboy27/tcp_server.git
``` 

#### В 1 терминале запустить сервер
```
python src/server.py
```
#### Во 2 терминале запустить тесты
```
python src/test_client.py
```
Результат будет в терминале
